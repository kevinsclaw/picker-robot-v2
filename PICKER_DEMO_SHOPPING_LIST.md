## 相机支架方案

### 方案 A: 铝型材 DIY（推荐 ✅）

| 物品 | 规格 | 参考价 | 链接关键词 |
|------|------|--------|-----------|
| 铝型材 | 2020 型材，长 60cm × 2根 + 40cm × 1根 | ~25元 | "2020铝型材 切割" |
| 直角连接件 | 2020 L型角码 × 4个 | ~8元 | "2020铝型材角码" |
| T型螺母+螺栓 | M5 × 若干 | ~5元 | "2020 T型螺母 M5" |
| 桌面夹 | G字夹或C型夹 2个 | ~10元 | "桌面G字夹 2寸" |
| 相机固定件 | 1/4螺丝转接板（相机通用） | ~5元 | "相机1/4螺丝固定板 铝型材" |
| **小计** | | **~53元** | |

搭建方式:
```
     ┌────── 40cm 横梁 ──────┐
     │         📷            │
  60cm 立柱              60cm 立柱
     │                      │
  ===桌夹==============桌夹===  ← 夹在桌边
```

### 方案 B: 现成悬臂支架（省事）

| 物品 | 规格 | 参考价 | 链接关键词 |
|------|------|--------|-----------|
| 相机悬臂支架 | 桌面夹持式，臂长50-80cm，可调角度 | ~40-80元 | "摄像头悬臂支架 桌面夹持" |
| 相机转接件 | 1/4螺丝（一般支架自带） | 0 | - |
| **小计** | | **~40-80元** | |

搭建方式:
```
          ╭──── 悬臂 ────📷
          │          (朝下拍)
    ══夹座══
       ↑
    夹在桌边
```

**我推荐方案 B** — 一根悬臂搞定，调角度方便，拆装快。淘宝搜"**俯拍支架 桌面夹**"或"**直播俯拍悬臂架**"，选臂长 ≥60cm 的，能调节角度的。
